O que é Unit Economics?
São as métricas que mostram se cada cliente é lucrativo para o negócio. As principais são:

• LTV (Lifetime Value): Quanto de receita um cliente gera durante todo o relacionamento com a empresa
• CAC (Custo de Aquisição de Cliente): Quanto custa para conquistar um novo cliente (marketing + vendas)
• LTV/CAC Ratio: Quanto você ganha para cada R$ 1 investido. Ideal: acima de 3x
• Payback: Tempo para recuperar o investimento em CAC. Ideal: menos de 12 meses
• Margem de Contribuição: Lucro após custos diretos (LLM, suporte, infraestrutura)
